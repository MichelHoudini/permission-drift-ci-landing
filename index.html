What it checks

Simple signals for permission drift

<all_urls>

Flags broad host access that can expand what an extension can read or change.

Manifest V2

Highlights Manifest V2 usage so teams can review legacy extension configuration.

eval / new Function

Detects dynamic code execution patterns that deserve careful review.

Remote scripts

Calls out script loading from remote URLs that can shift behavior outside the repo.

Problematic CSP

Reviews content security policy changes that may weaken extension protections.

Privacy drift

Surfaces privacy-impacting changes in permissions, data access, and disclosures.
